# SWE_101_02240349_A1# Campus Companion App

## Overview

Campus Companion is a mobile application developed using React Native (Expo) to assist new students in accessing essential campus information quickly and easily. The app provides a simple and user-friendly interface to view important contacts, class schedules, and campus-related resources.

## Installation & Running the App

1. Install dependencies:
   npm install

2. Start the Expo server:
   npx expo start

3. Run the app:

   * Press "a" for Android emulator
   * OR scan QR code using Expo Go app

## Known Issues / Limitations

* The schedule is static and not dynamically updated.
* Limited number of contacts included.
* Basic UI design (can be improved in future versions).
